Features
--------

* Conversation recurrence plotting
* Concordance search and export


Requirements
------------

* Python 3.7+
* spacy
* pandas
* Interactive widgets are designed for use in Jupyter Lab (3+)


Installation
------------

You can install *ant-widgets* via pip_ from PyPI_:

.. code:: console

   $ pip install ant-widgets
